border-width

  • thin = 1px
    • .thin { border-width: thin 1px; }
  • medium = 3px
    • .medium { border-width: medium 3px; }
  • thick = 5px（此处书中写的是4px）
    • .thick { border-width: thick 5px; }

border-style

  • none(default)
    • .none { border-style: none; }
  • solid
    • .solid { border-style: solid; }
  • dashed
    • .dashed { border-style: dashed; }
  • dotted
    • .dotted { border-style: dotted; }
  • double
    • .double { border-style: double; }
    值 双边边框表现规则 效果预览
    1px 0+1+0
    2px 1+0+1
    3px 1+1+1
    4px 1+2+1
    5px 2+1+2
    6px 2+2+2
    7px 2+3+2
  • 其他：inset、outset、groove、ridge

border-color

border-color默认颜色就是color色值
.border-color {
    border: 10px solid;
    color: red;
}

其他比较

  • 设置 border-width
    • .border-10px { border: 10px; }
  • 设置 border-color
    • .border-red { border: red; }
  • 设置 border-style
    • .border-solid { border: solid; }
